# Beampattern Shaping in 4D-Imaging Automotive MIMO Radars MATLAB Code

This repository contains MATLAB code developed to support the findings and methodologies presented in the paper titled:

**"Beampattern Shaping in 4D-Imaging Automotive MIMO Radars"**  
Published in *IEEE Transactions on Radar Systems*  
DOI: [10.1109/TRS.2024.3443301](https://doi.org/10.1109/TRS.2024.3443301)

The code is designed to reproduce the results and figures discussed in the paper, providing hands-on experience to understand and evaluate the proposed beampattern shaping method in automotive MIMO radar systems.

## 🚀 Overview

This repository includes tools for simulating and evaluating the beampattern shaping methodology in automotive MIMO radar systems for 4D imaging. The code allows users to generate beampatterns and visualize their performance, as described in the paper.

### Main Features:
- **Beampattern Shaping Method**: Code to implement the proposed beampattern shaping for MIMO radar in 4D imaging applications.
- **Automotive Radar Simulations**: Simulate MIMO radar systems designed for automotive 4D imaging tasks.
- **Result Reproduction**: Reproduce figures and results discussed in the paper.

## 🔥 Getting Started

### Results and Figures

To visualize the results and figures presented in the paper, follow these steps:
1. Navigate to the `results` folder.
2. Execute the script **`result.m`**. This script will generate the figures corresponding to the results discussed in the paper, allowing you to directly compare and analyze the methodology.

> **Note**: Figures 1, 2, and 16 are not included in the MATLAB implementation.

### Main Functionality

The core function of the proposed beampattern shaping method is contained in the **`main functions`** folder. Users interested in exploring the methodology further are encouraged to check out the code in this folder.
