# IEEE SPS 2024 - Radar Signal Processing Mastery: Course Materials

## Overview

This repository contains the materials for the **IEEE SPS 2024 Radar Signal Processing Mastery** short course, presented by **Mohammad Alaee-Kerahroodi**. The course focuses on the theoretical and practical aspects of radar signal processing with a particular emphasis on **mmWave MIMO radar** sensors.

Participants will be introduced to the fundamental concepts of **radar waveforms**, **signal processing techniques**, and **optimization methods** for radar systems. The course also covers **practical applications** and **hands-on exercises**, helping participants enhance their radar signal processing skills.

The materials provided include:

- **Presentation Slides**: Slides used during the course, covering topics such as radar signal processing theory and practical techniques.
- **MATLAB & Python Scripts**: Scripts related to the course's key topics and practical exercises.
- **Exercises and Quiz**: To test your understanding of radar signal processing concepts.

Access the full **IEEE SPS 2024 Radar Signal Processing Mastery Course** bundle here:  
[IEEE SPS 2024 Course Bundle](https://resourcecenter.ieee.org/education/courses/spscourse24002bundle)

## Course Content

The course covers the following topics:

1. **Introduction to Radar Signal Processing**
   - Overview of radar systems: SISO, phased array, and MIMO radar
   - Key radar parameters: Pulse Repetition Frequency (PRF), Doppler effect, and range resolution

2. **Radar Waveforms**
   - Fundamentals of radar waveforms: LFM, FMCW, PMCW
   - Waveform optimization techniques

3. **Signal Processing Techniques**
   - Matched filtering, pulse compression, and Doppler processing
   - Signal-to-noise ratio (SNR) optimization

4. **Advanced Radar Signal Processing**
   - MIMO radar systems: multiplexing and waveform diversity
   - Optimization for radar systems using non-convex techniques: gradient descent, majorization minimization

5. **Hands-on Exercises**
   - MATLAB and Python code examples for practical implementation
   - Exercises to apply theory to real-world radar systems

6. **Quiz**
   - A quiz designed to test the participants' understanding of radar signal processing principles and applications.

## Installation

To use the MATLAB and Python scripts, follow the steps below:

### MATLAB
1. Ensure that you have **MATLAB** installed (version 2020b or higher).
2. Clone or download this repository to your local machine.
3. Open the provided `.m` scripts in MATLAB and run them to view the results.
